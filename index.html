Propriedades mais usadas

Elementos em uma página da web são caixas retangulares com quatro camadas: Conteúdo, Preenchimentos(Padding), Bordas e Margens.

Qualquer elemento tem quatro espaçamentos internos contados no sentido horário, de cima para a direita, para baixo, para a esquerda.

Exemplo:


      img {
       padding-top: 20px;      /*Cima*/
       padding-right: 20px;    /*Direita*/
       padding-bottom: 20px;   /*Baixo*/
       padding-left: 20px;     /*Esquerda*/

        /*OU*/
        
        img{
          padding: 20px 20px 20px 20px;
          
            /*Cima, Direita, Baixo, Esquerda*/ 
        }


📌 color → cor do texto ️
📌 background-color → cor de fundo 🎆
📌 font-size → tamanho da fonte 🔠
📌 font-family → tipo da fonte (Arial, Verdana...) ✍️
📌 font-style → Texto de um elemento em itálico. Valor(italico;).
📌 font-weight: → Texto de um elemento em negrito. Valor(bold;).
📌 text-align → alinhamento (left, center, right) ↔️
📌 margin → espaço por fora do elemento 📏
📌 padding → espaço por dentro do elemento 📦
📌 border → borda (grossura, tipo, cor) ⬛
📌 width e height → largura e altura 📐 


p {
 color: red;                 /* cor do texto ️ */
 background-color: yellow;   /* cor de fundo 🎆 */
 font-size: 20px;            /* tamanho da fonte 🔠 */
 font-family: Arial, sans-serif; /* tipo da fonte ✍️ */
 font-style: italico;        /*Texto em itálico*/
 font-weight: bold;          /*Texto em negrito*/
 text-align: center;         /* alinhamento ↔️ */
 margin: 20px;               /* espaço por fora 📏 */
 padding: 10px;              /* espaço por dentro 📦 */
 border: 2px solid blue;     /* borda ⬛ */
 border-radius: 10px;        /* arredonda os cantos↗️*/
 width: 300px;               /* largura 📐 */
 height: 100px;              /* altura 📐 */
  
}

Frase descorada

Display: flex;

display: flex; é usada em CSS para transformar um elemento em um container flexível (ou flex container),
permitindo que seus elementos filhos (os itens flexíveis) sejam organizados de forma
mais eficiente e responsiva.

justify-content: ;

📌 flex-start → Os itens são alinhados ao lado esquerdo do contêiner.
📌 flex-end → Os itens são alinhados ao lado direito do contêiner.
📌 center → Os itens são alinhados no centro do contêiner.
📌 space-between → Os itens são exibidos com espaçamento igual entre eles.
📌 space-around → Os itens são exibidos com espaçamento igual ao redor deles.

flex-direction: ;

📌 row → Os itens são colocados na mesma direção do texto.
📌 row-reverse → Os itens são colocados opostos à direção do texto.
📌 column → Os itens são colocados de cima para baixo.
📌 column-reverse → Os itens são colocados de baixo para cima.


Class e id

📌 class → class é usado no HTML para marcar elementos que terão omesmo estilo.


📌 No CSS, você seleciona uma classe usando um ponto .
(ex: .minhaClasse)

📌 O atributo id serve para identificar um elemento único na página. No CSS, para selecionar um id, usamos a # (cerquilha).

⚠️ Importante: cada id deve ser único(não repetir em outros elementos).

HTML


    
<img class="planeta" src="https://mimo.app/r/earth.png">
  
 <p class="azul">Frase azul</p>
 <p class="azul">Frase 2 azul</p>
 <p>Frase normal</p>
 <p id="unica">Frase unica</p>
 <p id="unica2">Frase unica 2</p>

CSS


    .planeta{
       height: 100px;
       width: 100px;
    }
      .azul{
      color: blue;
    }
     #unica{
     color: green; 
    }
     #unica2{
     color: orange;
    }
